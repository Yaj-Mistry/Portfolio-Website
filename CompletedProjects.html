Snake

This is a snake prototype that I made a couple years ago. To make this I first
divided the canvasinto a grid with each cell being a 20 by 20 pixel square
and I lower the framerate as well. After that I made the snake itself, At first I was
confused on how to make the snake body,but then I realized, it should be an array.
The head of the snake is an independant object its just the tail. The tail is an array
where everytime the head makes contact with the apple, I add on to the array.
After I've added an element to the array I run a for loop and add a square
to the back of the snake for every element in the array.
